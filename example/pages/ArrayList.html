ArrayList

This is very similar to the Array that you are normally used to in JavaScript. This is a little different take though. Here are the available methods which we can expound upon in the future. And we will.
Method Description
pushAdd an element to the Array
popRemove the last element of the array
getGet an element at a given position in the array
deleteRemove and element at a given position in the array
_collapseToInternal function use to collapse all remaining items down in position by 1
 

        import { ArrayList } from "../../src";

        const al = new ArrayList();
        al.push({ id: 1, name: "mike" });
        al.push({ id: 2, name: "tommy" });
        al.push({ id: 3, name: "tj" });
        al.push({ id: 4, name: "ben" });
        al.push({ id: 5, name: "jimmy" });
        
        console.log(al);
Let's break this one down a little bit. We have pushed 4 people onto our array. It looks like this:
Now if we try and get someone at position 2, it is going to look like this:

        console.log(al.get(2))
Notice that it is actually the record with id 3. That is becuase this is a zero indexed array. Now here is what our array looks like it we run the pop method on the array. 

        al.pop();
        console.log(al);
Notice that jimmy, who was the last element is now gone. Now we will delete array id of two and see what that looks like.

        al.delete(2);
        console.log(al);
Notice that tj is now gone. Here is the complete source code for this page.
